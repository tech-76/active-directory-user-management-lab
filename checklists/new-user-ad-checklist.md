# New User Active Directory Checklist

## Request Details

- [ ] Full name confirmed
- [ ] Username format confirmed
- [ ] Job title confirmed
- [ ] Department confirmed
- [ ] Manager confirmed
- [ ] Start date confirmed
- [ ] Required groups confirmed
- [ ] Required shared resources confirmed
- [ ] Approval received

## Account Creation

- [ ] Checked for duplicate account
- [ ] Created user account
- [ ] Set display name
- [ ] Set username
- [ ] Set temporary password
- [ ] Required password change at next sign-in
- [ ] Placed user in correct OU
- [ ] Added approved group memberships

## Access Confirmation

- [ ] Security groups confirmed
- [ ] Shared resource access confirmed if applicable
- [ ] Account enabled according to start date policy
- [ ] User instructions prepared
- [ ] Ticket updated

## Final Checks

- [ ] No password included in ticket notes
- [ ] No unnecessary access assigned
- [ ] Requester notified
- [ ] Ticket closed with clear resolution
