# Password Reset Checklist

## Verification

- [ ] User identity verified
- [ ] Request reviewed
- [ ] Account found in Active Directory
- [ ] Account status checked
- [ ] Lockout status checked

## Reset Steps

- [ ] Password reset according to policy
- [ ] Password change required at next sign-in
- [ ] User informed through approved method
- [ ] User tested sign-in
- [ ] User confirmed access restored

## Security Checks

- [ ] No password written in ticket notes
- [ ] Suspicious activity reviewed if needed
- [ ] Possible compromise escalated if needed
- [ ] Repeated failures reviewed if needed

## Ticket Closure

- [ ] Troubleshooting steps documented
- [ ] Resolution documented
- [ ] User confirmation documented
