# Account Lockout Checklist

## Initial Questions

- [ ] When did the lockout start?
- [ ] Did the user recently change password?
- [ ] Is the user signed in on multiple devices?
- [ ] Does the user use Outlook on mobile?
- [ ] Does the user use VPN?
- [ ] Does the user have mapped drives?
- [ ] Does the lockout keep happening?

## Account Checks

- [ ] User identity verified
- [ ] Account found in Active Directory
- [ ] Account lockout confirmed
- [ ] Account unlocked if approved
- [ ] Password reset if required

## Common Sources of Old Passwords

- [ ] Mobile email app
- [ ] Outlook desktop
- [ ] VPN client
- [ ] Wi-Fi profile
- [ ] Mapped network drive
- [ ] Remote desktop saved credentials
- [ ] Windows Credential Manager
- [ ] Browser saved password

## Escalation

- [ ] Repeated lockouts
- [ ] Unusual sign-in times
- [ ] Privileged account involved
- [ ] Multiple users affected
- [ ] Possible attack or compromise

## Closure

- [ ] Cause or likely cause documented
- [ ] User confirmed access restored
- [ ] Follow-up recommended if lockout returns
