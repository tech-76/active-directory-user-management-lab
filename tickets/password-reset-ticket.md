# Ticket: Active Directory Password Reset

## Ticket Summary

User requested a password reset after forgetting their password.

## User Impact

User could not sign in to their workstation or access network resources.

## Reported Issue

User reported they forgot their password and could not log in.

## Troubleshooting Steps

1. Verified user identity according to support procedure.
2. Confirmed account was active.
3. Confirmed account was not disabled.
4. Reset password according to policy.
5. Required password change at next sign-in.
6. Instructed user to sign in.
7. Confirmed user created a new password.
8. Confirmed user could access workstation and network resources.

## Resolution

Password reset completed successfully. User confirmed access was restored.

## Customer Communication

Your password has been reset according to policy. Please sign in and create a new password when prompted.

## Final Ticket Note

Verified user identity, reset AD password, required password change at next sign-in, and confirmed successful login.
