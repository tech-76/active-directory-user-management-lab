# Ticket: New User Active Directory Account

## Ticket Summary

Manager requested a new Active Directory account for an incoming employee.

## User Impact

New employee required account access before their start date.

## Request Details

- New user account required
- Department access required
- Security group access required
- Start date provided by manager
- Approval provided by manager

## Troubleshooting / Fulfillment Steps

1. Reviewed new user request.
2. Confirmed approval.
3. Confirmed username format.
4. Checked for duplicate account.
5. Created Active Directory user account.
6. Placed user in correct department OU.
7. Set temporary password according to policy.
8. Required password change at first sign-in.
9. Added approved security groups.
10. Documented account setup.

## Resolution

New user account was created and assigned to the approved groups.

## Final Ticket Note

Created approved Active Directory account for new user, placed account in correct OU, assigned approved group memberships, and documented onboarding completion.
